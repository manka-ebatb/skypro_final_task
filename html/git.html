Git

  • git branch — отображает список локальных веток проекта.
  • git checkout — переключается на указанную ветку.
  • git merge — объединяет выбранную ветку с текущей, добавляя её изменения.
  • git add — добавляет файлы в индекс (подготовку для коммита).
  • git commit — сохраняет изменения в истории коммитов.
